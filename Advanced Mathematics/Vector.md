# Vector
[Vector Space](https://en.wikipedia.org/wiki/Vector_space)
```


  B---------C
 /         /
O-------->A

->   ->    ->
OB + OA  = OC


O---->A(O)--------->B               

A<----O---------->B           

Assume:
    
    O------A    

    ->    ->            ->    ->
    a   = OA   then   - a   = AO   // 

Then:
    ->     ->     ->    ->
    a   +  b  =   b   + a
    ->  ->  ->    ->  ->   ->   ->   ->  -> 
    a + b + c  = (a + b) + c  = a + (b + c)

```

```
   ->           ->
|λ a|   = λ  * |a|    , λ > 0

  ->    ->
λ a   = 0     , λ = 0

   ->          ->
|λ a| = |λ| * |a|

```

```
   ->    ->       ->                 ->            ->   ->
```
