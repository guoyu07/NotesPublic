网页特殊符号与ASCII编码在线转换

美国信息互换标准代码 (American Standard Code for Information Interchange, ASCII) 是基于拉丁字母的一套电脑编码系统。它主要用于显示现代英语和其他西欧语言。它是现今最通用的单字节编码系统，并等同于国际标准ISO/IEC 646。

对于很多特殊字符，如中文，或者如下表特殊字符集，经常在不同的国家的浏览器下会出现乱码。就算用UTF-8编码，对于特殊的情况下，对于两种不同的编码交互使用也会出现乱码。所以对一些符号进行ASCII编码处理，或者HTML实体处理是最好的实现兼容性的方式。因为HTML实体或者ASCII全部使用单字节英文字母表示，在任何编码、任何浏览器下都能很好的显示而不会出现乱码。

JS利用charCodeAt()转换字符串为ASCII编码

var s="E维科技，专注网络营销！";
var k=s.split("");
//JS拆分中英文数字混合字符串
var x="";
var i=0;
for(i;i<k.length;i++)
   {
    x+="&#"+k[i].charCodeAt()+";";
   }
alert(s+"转换成ASCII编码是："+x);
//注意使用alert，如果使用document.write就只能查看源码才能看到转换后的情况
