ADO Direction 属性

返回 ADO Parameter 对象参考手册

定义和用法

Direction 属性可设置或返回一个 ParameterDirectionEnum 值，指示 Parameter 所表示的是输入参数、输出参数还是两者都表示，或者指示参数是否是存储过程的返回值。默认值是 adParamInput。

Direction 属性为读/写，它允许用户使用不返回此信息的提供者，或者在不希望 ADO 为检索参数信息而额外调用提供者时设置此信息。

并非所有提供者都能在其存储过程中确定参数的方向。在这种情况下，必须在执行查询前设置 Direction 属性。

语法

objparameter.Direction

实例

<%
set comm=Server.CreateObject("ADODB.Command")
set para=Server.CreateObject("ADODB.Parameter")
para.Type=adVarChar
para.Size=25
para.Direction=adParamInput
para.Value=varfname
comm.Parameters.Append para
%>

ParameterDirectionEnum Values

常量 值 描述
adParamUnknown 0 指示该参数的方向未知。
adParamInput 1 默认值。指示该参数是输入参数。
adParamInputOutput 3 指示该参数既是输入参数，又是输出参数。
adParamOutput 2 指示该参数是输出参数。
adParamReturnValue 4 指示该参数是返回值。
