ADO Delete 方法

返回 ADO Parameter 对象参考手册

定义和用法

Delete 方法可从 Parameters 集合中删除某个对象。

注释：此方法只能用在 Command 对象的 Parameters 集合上。

在调用 Delete 方法时您必须使用 Parameter 对象的 Name 属性或其集合索引 ―― 对象变量不是有效的参数。

语法

objparameter.Delete index
参数 描述
index String 值，该值包含要删除的对象的名称，或者对象在集合中的顺序位置 (索引）。
