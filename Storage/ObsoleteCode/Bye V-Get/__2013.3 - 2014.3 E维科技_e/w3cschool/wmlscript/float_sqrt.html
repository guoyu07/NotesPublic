WMLScript sqrt() 函数

返回 WMLScript Float 库参考手册

sqrt() 函数返回一个数的平方根。

语法

n = Float.sqrt(x)
成分 描述
n 该函数返回的浮点数。
x 数字。

例子

var a = Float.sqrt(9);
var b = Float.sqrt(6.25);

结果

a = 3
b = 2.5
