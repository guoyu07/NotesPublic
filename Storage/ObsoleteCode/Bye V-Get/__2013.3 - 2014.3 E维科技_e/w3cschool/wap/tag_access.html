WML <access> 标签

返回 WML 参考手册

定义和用法

<access> 标签定义关于 WML 卡片组的访问控制的信息。

注释：如果 <access> 标签没有设置，任何其他 deck/card 都能够访问此 deck。

注释：<access> 标签永远位于 <head> 标签内部。每个 deck 只能有一个 <access> 标签。

语法

<head> 
<access/> 
</head>

属性

属性 值 描述
domain cdata 能够访问此 deck 中 card 的其他 deck 的 URL。
path cdata 能够访问此 deck 中 card 的其他 deck 的 URL。
class cdata 为此元素设置的 class 名。这个 class 名对大小写敏感。一个元素可链接多个 class。claas 属性中的多个 class 名通过空格进行分隔。
id id 为此元素设置的唯一名称。

实例

访问被限制于 e.v-get.com 中的其他 deck/card：

<head>
 <access domain="e.v-get.com"/>
</head>

访问被限制于 e.v-get.com/wap 中的其他 deck/card：

<head>
 <access domain="www.w3School.com.cn" path="/wap"/>
</head>
