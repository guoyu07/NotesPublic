ASP.NET HtmlImage 控件

返回 HTML 服务器控件参考手册

定义和用法

HtmlImage 控件用于控制 <img> 元素。在 HTML 中，<img> 元素用于显示图像。

属性

属性 描述
Align

如何根据周围的元素排列图像。合法的值有：

  • top
  • middle
  • bottom
  • left
  • right
Alt 关于图像的简短描述。
Attributes 返回该元素的所有属性名称和值对。
Border 图像周围的边框的宽度。
Disabled 布尔值，指示是否禁用该控件。默认是 false。
Height 图像的高度。
id 控件的唯一 id。
runat 规定该控件是一个服务器控件。必须被设置为 "server"。
Src 要显示的图像的 URL。
Style 设置或返回被应用到该控件的 CSS 属性。
TagName 返回元素的标签名。
Visible 布尔值，指示该控件是否可见。
Width 图像的宽度。
