CSS outline-color 属性

返回 CSS 参考手册

定义

outline (轮廓）是绘制于元素周围的一条线，位于边框边缘的外围，可起到突出元素的作用。outline 属性可设置元素周围的轮廓线。

注释：请始终在 outline-color 属性之前声明 outline-style 属性。元素只有获得轮廓以后才能改变其轮廓的颜色。

注释：轮廓线不会占据空间，也不一定是矩形。

继承性：No

说明

outline-color 属性设置一个元素整个轮廓中可见部分的颜色。要记住，轮廓的样式不能是 none，否则轮廓不会出现。

JavaScript 语法

CSS 属性也可通过一段 JavaScript 被动态地改变。

脚本语法：

object.style.outlineColor="#00FF00"

在我们的 HTML DOM 教程中，您可以找到更多有关 outlineColor 属性 的细节。

在我们的 HTML DOM 教程中，您也可以找到完整的 Style 对象参考手册。

例子

button
{ 
outline-style: solid;
outline-color: red;
}

可能的值

值 描述
invert 默认。执行颜色反转 (逆向的颜色）。这么做是为了使边框在不同的背景颜色中都是可见的。
color 可以是颜色名 (如 red)，rgb 值(如 rgb(255,0,0)) 或十六进制值 (如 #ff0000)。
