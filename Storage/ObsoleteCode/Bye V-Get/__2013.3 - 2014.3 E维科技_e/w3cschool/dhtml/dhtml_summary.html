DHTML 概要

本教程已经向您讲解了如何使用 DHTML 来创建动态性和交互性更强调网站。

您已经学习到如何通过 HTML、CSS 以及 JavaScript 使用 HTML 文档更有动感。

如需更多有关 DHTML 方面的知识，请看参阅我们的 DHTML 实例 和 HTML DOM 参考手册。

您已经学习了 DHTML，下一步呢？

下一步应该学习 HTML DOM 和 ASP。

HTML DOM

HTML DOM 定义了一套访问和处理 HTML 文档的标准方法。

HTML DOM 独立于平台和语言，可被诸如 Java, JavaScript 以及 VBScript 等编程语言使用。

如果您希望学习更多有关 DOM 的知识，请访问我们的 HTML DOM 教程。

ASP

与一个 HTML 文件中的脚本在客户机 (浏览器）中执行不同，APS 文件中的脚本在服务器上执行。

通过 ASP，您能够动态地编辑、修改或添加一张网页的任何内容，对被提交自 HTML 表单的数据进行响应，访问任何数据或数据库并向浏览器返回结果，为不同的用户定制更具价值的页面。

由于 ASP 文件以纯粹的 HTML 返回结果，它们可通过任何浏览器来查看。

如果您希望学习更多有关 ASP 的知识，请访问我们的 ASP 教程。
