XML DOM createAttributeNS() 方法

返回 Document 对象参考手册

定义和用法

创建具有指定的名称和命名空间的 Attr 节点。

语法：

createAttributeNS(uri,name)
参数 描述
uri Attr 的命名空间的唯一标识符。如果没有命名空间。则为 null。
name 属性的限定名，应包括命名空间前缀、冒号和本地名。

返回值

返回新创建的 Attr 节点，具有指定的名称和命名空间。

说明

createAttributeNS() 方法与 createAttribute() 方法相似，只是它创建的 Attr 节点除了具有指定的名称外，还具有指定的命名空间。只有使用命名空间的 XML 文档才会使用该方法。
