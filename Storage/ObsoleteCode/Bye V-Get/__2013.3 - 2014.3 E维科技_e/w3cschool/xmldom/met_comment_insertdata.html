XML DOM insertData() 方法

返回 Comment 对象参考手册

定义和用法

insertData() 方法把字符串插入注释节点。

语法：

commentNode.insertData(start,string)
参数 描述
start 必需。规定再何处插入字符。该值从 0 开始。
length 必需。要插入的字符的数量。

描述

该方法把字符串 string 插入指定位置 length 的 Comment 节点的文本处。

实例

以下代码段使用 JavaScript 函数 loadXMLDoc() 把 XML 文件 books_comment.xml 载入 xmlDoc 中，然后向第一个 Comment 节点插入一个字符串：

xmlDoc=loadXMLDoc("books_comment.xml");
x=xmlDoc.getElementsByTagName("book")[0].childNodes;
for (i=0;i<x.length;i++)
{ 
  if (x[i].nodeType==8)
  { 
  //仅仅处理 comment 节点
  x[i].insertData(10,"Illustrated ");
  document.write(x[i].data);
  document.write("<br />");
  } 
}

以上代码的输出：

(Book 6) (Illustrated Hardcover)

在本例中，我们使用一段循环和 if 语句来执行只针对 comment 节点的处理。comment 节点的节点类型是 8。
