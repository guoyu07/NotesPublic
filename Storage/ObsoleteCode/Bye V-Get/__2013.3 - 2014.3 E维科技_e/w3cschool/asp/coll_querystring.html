ASP QueryString 集合

返回 Request 对象参考手册

定义和用法

QueryString 集合用于取回 HTTP 查询字符串中的变量值。

HTTP 查询字符串 (HTTP query string) 是由问号 (?) 之后的值规定的，比如：

<a href="test.asp?txt=this is a query string test">带有查询字符串的链接</a>

上面的代码可生成一个名为 txt 且值为 "this is a query string test"的变量。

查询字符串同样可以通过表单提交来生成，或者通过用户在浏览器的地址栏中输入查询。

语法

Request.QueryString(variable)[(index)|.Count]
参数 描述
variable 必需。在 HTTP 查询字符串中要取回的变量名称。
index 可选。为一个变量规定多个值之一。从 1 到 Request.QueryString(variable).Count

实例

例子 1

遍历查询字符串中所有变量 n 的值：

假设，这是被送出的请求：

http://e.v-get.com/test/names.asp?n=John&n=Susan

而 names.asp 包含以下代码：

<%
for i=1 to Request.QueryString("n").Count 
  Response.Write(Request.QueryString("n")(i) & "<br />")
next
%>

文件 names.asp 会显示出：

John
Susan

例子 2

假设，这是被送出的字符串：

http://e.v-get.com/test/names.asp?name=John&age=30

上面的代码产生了以下 QUERY_STRING 值：

name=John&age=30

现在，我们可以通过脚本使用这些信息：

Hi, <%=Request.QueryString("name")%>. 
Your age is <%= Request.QueryString("age")%>.

输出：

Hi, John. Your age is 30.

假如您没有规定任何要显示的变量值，比如这样：

Query string is: <%=Request.QueryString%> 

输出将成为这样：

Query string is: name=John&age=30
