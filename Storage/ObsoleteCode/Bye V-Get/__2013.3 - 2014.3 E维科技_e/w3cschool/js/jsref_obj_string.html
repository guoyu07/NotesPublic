JavaScript String 对象参考手册

字符串是 JavaScript 的一种基本的数据类型。

String 对象的 length 属性声明了该字符串中的字符数。String 类定义了大量操作字符串的方法。

需要注意的是，JavaScript 的字符串是不可变的，String 类定义的方法都不能改变字符串的内容。
