JavaScript Number 对象

JavaScript Number 对象无法设置具体的数字类型 (比如 integer, short, long 或 double）。在 JavaScript 中，所有数字都是由浮点数表示的。JavaScript 采用 IEEE 754 标准定义的 64bit 浮点格式表示数字。这意味着它能表示的最大值是 1.7976931348623157e+308 (正的)，最小值是 5e-324 (负的)。

请访问我们的 JavaScript Number 对象参考手册，获得有关所有 JavaScrip Number 对象的属性和方法的完整参考信息。

该手册提供了每种属性和方法的详细描述和代码实例！
