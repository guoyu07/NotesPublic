SMIL 向 web 增加了对定时和媒介同步的支持。

SMIL

SMIL (Synchronized Multimedia Integration Language) 被设计为用来启用 web 上的多媒体呈现。

SMIL 呈现可由音频、视频、图像、文本以及其他媒介类型组成。

SMIL 是一门基于 XML 的类 HTML 语言。

您可以在我们的 SMIL 教程 中阅读更多有关 SMIL 的内容。

HTML+TIME

HTML+TIME 指的是对 HTML 的定时多媒体交互扩展 (Timed Interactive Multimedia Extensions for HTML）。它的作用是向 HTML 添加 SMIL 1.0 的定时和同步支持。

HTML+TIME 被以下单位提交到 W3C ：Microsoft、Macromedia、Compaq/Digital 以及 Digital Renaissance。

您可以在我们的 SMIL 教程 教程中阅读更多有关 HTML+TIME 的知识。

HTML+SMIL

HTML+SMIL (对 HTML+TIME 的修正）曾在一份早期的 SMIL 2.0 工作草案被提及。

SMIL 2.0 成为 W3C 推荐标准后，HTML+SMIL 被从 SMIL 2.0 中删除，并成为名为 XHTM+SMIL 的独立工作草案。

XHTML+SMIL

XHTML+SMIL 在 XHTML 中提供了对 SMIL 2.0 功能性的支持，比如动作、媒介、定时、同步以及直接转换。

XHTML+SMIL 目前是被提交到 W3C 的一份记录，其意图是 XHTML+SMIL 能够被用作向 XHTML 中整合 SMIL 的工作基础。

您可以在我们的 SMIL 教程 教程中阅读更多有关 XHTML+SMIL 的内容。

HTML/XHTML+SMIL 的历史

15. June 1998
SMIL 1.0 成为 W3C 推荐标准。
18. September 1998
作为一项向 HTML 增加了 SMIL 1.0 定时和同步的提议，HTML+TIME 被 Microsoft、Macromedia、Compaq/Digital 以及 Digital Renaissance 提交到 W3C。
25. February 2000
HTML+TIME 以重新命名的 HTML+SMIL 被添加到 SMIL 2.0 的工作草案。
22. June 2000
HTML+SMIL 被从 SMIL 2.0 工作草案中删除。
07. August 2001
SMIL 2.0 成为 W3C 标准。
07. August 2001
HTML+SMIL 成为一个独立的工作草案，被重新命名为 XHTML+SMIL。
31. January 2002
XHTML+SMIL 作为 W3C 记录被重新提交。
