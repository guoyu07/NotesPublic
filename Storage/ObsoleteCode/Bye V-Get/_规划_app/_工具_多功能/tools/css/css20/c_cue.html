cue-before版本：CSS2　 兼容性：无　继承性：无
语法：
cue : cue-before || cue-after
取值：
该属性是复合属性。请参阅各参数对应的属性。
说明：
设置在对象前后播放的声音。
假如只给出了一个值，它将用于 cue-before 属性和 cue-after 属性。假如给出了两个值，则第一个用于 cue-before 属性，第二个用于 cue-after 属性。
对应的脚本特性为 cue 。
示例：
div { cue: url(orchestralsneeze.wav) none }
div { cue: url(orchestralsneeze.wav); }
