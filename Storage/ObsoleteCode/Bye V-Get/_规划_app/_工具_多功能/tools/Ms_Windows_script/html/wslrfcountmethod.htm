Count 方法

返回 WshNamed 或 WshUnnamed 对象中的开关数。

object.Count

参数

object
Arguments 对象。

说明

Count 方法返回整数值。Count 方法主要是为 VBSript 用户所用, JScript 用户一般使用length 属性。

示例 (WshNamed)

下面的示例使用 WshNamed 对象演示 Count 方法。先在命令提示符处键入下面的文本：

myScript.vbs /c:"WSH is a wonderful thing" /s:"scripts are wonderful"

接着，添加下面的 VBScript 代码：

For i = 0 to WScript.Arguments.Count-1
    WScript.Echo WScript.Arguments.Named(i)
next i

生成的结果如下所示：

WSH is a wonderful thing
scripts are wonderful

示例 (WshUnnamed)

下面的示例使用 WshUnnamed 对象演示 Count 方法。先在命令行中键入下面的文本：

myscript.vbs "WSH is a wonderful thing" "scripts are wonderful"

接着，添加下面的 VBScript 代码：

For i = 0 to WScript.Arguments.Count-1
    WScript.Echo WScript.Arguments.Unnamed(i)
next i

结果如下所示：

WSH is a wonderful thing
scripts are wonderful
