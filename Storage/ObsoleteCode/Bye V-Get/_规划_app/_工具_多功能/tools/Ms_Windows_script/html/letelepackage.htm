Windows 脚本部件  

<package> 元素

包含多个脚本部件的定义。

<package>
      此处为一个或多个脚本部件
</package>

说明

当一个 .wsc 文件只包含一个脚本部件定义时，<package> 元素是可选的。

示例

<package>
<component id="MyScriptlet">
   (此处为脚本部件信息)
</component>

<component id="MyOtherScriptlet">
   (此处为脚本部件信息)
</component>
</package>
