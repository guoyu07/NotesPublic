<?XML?>元素

指明文件应分析为 XML。

<?XML version="version" [standalone="DTDflag"] ?>

参数

version
n.n 形式的字符串，指定文件的 XML 级别。使用值 1.0。
DTDflag
可选。布尔值，指明 XML 文件中是否包括对外部文档类型定义 (DTD) 的引用。脚本组件 XML 文件不包括这样的引用，因此该属性值始终是 "yes"。

说明

该声明必须是文件中的第一个元素，并且它的前面不能有任何空行。

如果有这个声明，就会令脚本组件编译器进入严格的 XML 模式，在该模式下，元素类型和属性名称是区分大小写的，属性值用单引号和双引号括起来，并且所有元素都要被分析。如果不包括这个声明，编译器就会允许使用较为宽松的语法。

如果要在支持 XML 的编辑器中编辑脚本组件文件，则应该包括该声明并遵循 XML 约定。

示例

下面的示例将演示 <?XML?>元素的用法：

<?XML version="1.0" standalone="yes" ?>
