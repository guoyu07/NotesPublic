Windows 脚本部件  

<?component?>

指定用于错误处理的属性。

<?component error="flag" debug="flag" ?>

值

flag
为一个布尔值： “true” 或 “false”、 1 或 0、或者是 “yes” 或 “no” 。所有属性的缺省值都为 false。

说明

由于脚本部件是作为 COM 部件来使用的，它们通常都是静静地运行。不过，在开发脚本部件时，如果能告知文件中的错误将是很有用的。您可以指定错误通知的类型：

  • error   将其设置为真，将显示脚本部件中的语法或运行错误的详细错误信息。
  • debug   将其设置为真将启用调试功能。如果没有启用该调试功能，则任何方式下都不能对脚本部件使用脚本调试器。

示例

<?component error="true" debug="true" ?>
