需要 Function 对象

试图对一个不是 Function 对象调用 Function prototype 方法，或者在函数调用上下文中使用了对象。例如，下面的代码将产生该错误，因为 example 不是一个函数。

var example = new Object();  // 创建一个称为 "example" 的新对象。
var x = example();  // 试图将 example 作为一个函数加以调用。

要改正此错误

  • 只对 Function 对象调用 Function prototype 方法。
  • 确保只使用函数调用运算符() 来调用函数。
