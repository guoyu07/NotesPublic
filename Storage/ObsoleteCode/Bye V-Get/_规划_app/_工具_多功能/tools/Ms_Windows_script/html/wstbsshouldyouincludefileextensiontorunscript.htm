运行脚本需要包括的内容

通过键入脚本名称运行脚本时所要键入的信息取决于所运行的 Windows 版本和脚本的运行方式。

WSH 可执行文件 Windows 版本 包括
命令提示符 Windows NT 或 2000 指定不带文件扩展名的脚本名称。例如：myScript。

注意   如果指定 WSH 可执行文件的名称，还须包括脚本文件的扩展名。例如：cscript myScript.wsf。
命令提示符 Windows 9x 或 Millennium 指定脚本文件的扩展名，并在脚本名称之前冠以 WSH 可执行文件的名称。

例如：cscript myScript.wsf。
“打开”对话框的“运行”命令 Windows NT、2000、9x 或 Millennium 指定脚本文件的扩展名。
