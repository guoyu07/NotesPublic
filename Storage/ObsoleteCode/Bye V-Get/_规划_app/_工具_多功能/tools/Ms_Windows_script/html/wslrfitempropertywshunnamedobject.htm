Item 属性 (WshUnnamed)

返回使用从零开始的索引的项目。

Object.Item(key)

参数

Object
WshUnnamed 对象。
key
与某个集合或 WshUnnamed 对象相关联。

说明

Item 属性返回一个字符串。对于集合来说，它将根据指定的关键字返回一个项。

对于未命名的对象，项没有命名，因此无法传递 WshUnnamed 参数的名称。

WshUnnamed 参数项按其包含在命令行中的顺序进行保存。

在命令行输入 WshUnnamed 参数时，如果要把字符串用引号括起来，那么可以在字符串参数中使用空格。

WshUnnamed 集合中的引号将被删除。

示例

下面的示例演示 Item 属性如何用于返回 WshUnnamed 集合中的项目。在本例中，提供了两个用来运行脚本的未命名参数。在脚本内部，代码可导致未命名参数输出。Item 属性用于索引未命名参数集。

' 在命令提示符处键入以下命令行运行该脚本。
myScript.vbs arg1 arg2

' 当以下代码在脚本内部执行时：
WScript.Echo WScript.Arguments.Unnamed.Item(0)
WScript.Echo WScript.Arguments.Unnamed.Item(1)

' 生成以下输出：
arg1
arg2
