Round 函数

返回按指定位数进行四舍五入的数值。

Round(expression[, numdecimalplaces])

参数

Expression
必选项。数值表达式被四舍五入。
Numdecimalplaces
可选项。数字表明小数点右边有多少位进行四舍五入。如果省略，则 Round 函数返回整数。

说明

下面的示例利用 Round 函数将数值四舍五入到两位小数：

Dim MyVar, pi
pi = 3.14159
MyVar = Round(pi, 2) 'MyVar contains 3.14。
