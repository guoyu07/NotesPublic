除法运算符 (/)

两个数值相除并返回以浮点数表示的结果。

result = number1/number2

参数

result
任意数值变量。
number1
任意数值表达式。
number2
任意数值表达式。

说明

如果一个或两个表达式为 Null 表达式，result 为 Null。任何表达式为 Empty 时，则按 0 处理。

要求

版本 1
