Month 函数

返回 1 到 12 之间的一个整数（包括 1 和 12），代表一年中的某月。

Month(date)

date 参数是任意可以代表日期的表达式。如果 date 参数中包含 Null，则返回 Null。

说明

下面的示例利用 Month 函数返回当前月：

Dim MyVar
MyVar = Month(Now) ' MyVar 包含
                   ' 与当前月对应的数字。

要求

版本 1
