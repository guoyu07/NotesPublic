Join 函数

返回一个字符串，此字符串由包含在数组中的许多子字符串连接创建。

Join(list[，delimiter])

参数

list
必选项。包含要连接的子字符串一维数组。
Delimiter
可选项。在返回字符串中用于分隔子字符串的字符。如果省略，将使用空字符 ("")。如果 delimiter 是零长度字符串，则在同一列表中列出全部项，没有分界符。

说明

下面的示例利用 Join 函数联合 MyArray 的子字符串：

Dim MyString
Dim MyArray(3)
MyArray(0) = "Mr."
MyArray(1) = "John "
MyArray(2) = "Doe "
MyArray(3) = "III"
MyString = Join(MyArray) 'MyString 包含 "Mr. John Doe III"。
