Windows 脚本宿主  

Execute 方法

开始执行远程脚本对象。

object.Execute

参数

object
WshRemote 对象

说明

WshRemote 对象的 Start 事件在脚本开始执行时引发。不要混淆 Execute 和 Exec 方法（后者属于 WScript 对象）。

示例

下面的示例演示 Execute 方法如何用于创建 WshRemote 对象（启动远程脚本的实例）。

[VBScript] 
Dim Controller, RemoteScript
Set Controller = WScript.CreateObject("WSHController")
Set RemoteScript = Controller.CreateScript("remote1.js")
RemoteScript.Execute

Do While RemoteScript.Status <> 2 
    WScript.Sleep 100
Loop
[JScript] 
var Controller = WScript.CreateObject("WSHController");
var RemoteScript = Controller.CreateScript("remote1.js");
RemoteScript.Execute();

while (RemoteScript.Status != 2) {
    WScript.Sleep(100);
}
