Windows 脚本宿主  

<object> 元素

定义 Windows 脚本 (.wsf) 文件中可由脚本引用的对象。

<object id="objID" [classid="clsid:GUID" | progid="progID"] />

参数

objID
必需。用于引用脚本中该对象的名称。对象 ID 值必须以字母打头，可以包含字母、数字和下划线 (_)。对象 ID 在整个 Windows 脚本文件的范围内必须是唯一的。
GUID
可选。对象的类 ID (GUID)。
progID
可选。对象的程序 ID，可指定它来替换类 ID。

说明

<object> 元素提供一种方法，以便在不使用诸如 CreateObject() 之类的函数的情况下，展示供 Windows 脚本文件中的脚本全局使用的对象。使用 <object> 元素可使对象用于全局范围并启用脚本工具，以便为对象的成员提供语句完成。

必须指定 classid 或 progid。

示例

<job>
<obect id="fso" progid="Scripting.FileSystemObject"/>
<script language="Jscript">

var a = fso.CreateTextFile("c:\\testfile.txt", true);
a.WriteLine("This is a test.");
a.Close();

</script>
</job>
