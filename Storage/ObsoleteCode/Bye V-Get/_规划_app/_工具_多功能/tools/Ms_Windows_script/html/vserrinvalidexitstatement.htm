无效 'exit' 语句

在以下程序控制结构之一中使用了Exit 语句：

  • With 语句。
  • If...Then...Else语句。
  • Select Case语句。

仅在如下结构中可以合法使用Exit 语句来退出：

  • Do...While 循环         (Exit Do)。
  • For...Next 循环         (Exit For)。
  • For Each...Next 循环   (Exit For)。
  • Function 过程      (Exit Function)。
  • Sub 过程         (Exit Sub)。

要纠正该错误

  • 可能不适当地使用了Exit If， Exit With，或Exit Select。
  • 不要和If， With，Select 语句一起使用Exit。
