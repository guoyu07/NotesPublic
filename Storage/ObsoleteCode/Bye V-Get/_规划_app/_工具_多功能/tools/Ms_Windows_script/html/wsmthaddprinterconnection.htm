AddPrinterConnection 方法

将基于 MS-DOS 的远程打印机连接添加到计算机系统中。

object.AddPrinterConnection(strLocalName, strRemoteName[,bUpdateProfile][,strUser][,strPassword])

参数

object
WshNetwork 对象。
strLocalName
表示指派给所连接打印机的本地名称的字符串值。
strRemoteName
表示远程打印机名称的字符串值。
bUpdateProfile
可选。表示打印机映射是否存储在用户配置文件中的布尔值。如果提供了 bUpdateProfile 且其值为 true，则该映射存储在用户配置文件中。默认值为 false。
strUser
可选。表示用户名的字符串值。如果要使用当前用户以外的用户配置文件映射远程打印机，则可指定 strUser 和 strPassword。
strPassword
可选。表示用户密码的字符串值。如果要使用当前用户以外的用户配置文件映射远程打印机，则可指定 strUser 和 strPassword。

说明

AddPrinterConnection 方法将网络打印机添加到 MS-DOS 打印机端口，如 LPT1。不能利用该方法添加基于 Windows 的远程打印机连接。要添加基于 Windows 的远程打印机连接，请使用 AddWindowsPrinterConnection 方法。

示例

以下代码使用 AddPrinterConnection 方法将网络打印机连接到 LPT1。

[VBScript] 
Set WshNetwork = WScript.CreateObject("WScript.Network")
WshNetwork.AddPrinterConnection "LPT1", "\\Server\Print1"
[JScript] 
var WshNetwork = WScript.CreateObject("WScript.Network");
WshNetwork.AddPrinterConnection ("LPT1", "\\\\Server\\Print1");
