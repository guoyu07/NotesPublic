无效 'for' 循环控制变量

创建了一个For...Next循环。换句话说，可能在For...Next中使用了一个非法的控制变量。下面的代码演示了一个For...Next 循环的正确结构。

For counter = start To end [Step step]
    [statements]
    [Exit For]
    [statements]
Next

要纠正该错误

修改For...Next 循环的控制变量，使其在外循环的范围内唯一。
