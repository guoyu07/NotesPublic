Windows 脚本部件  

创建脚本部件

创建 Windows® 脚本部件 (.wsc) 文件与创建任何包含脚本的文件（包括 HTML 文件）类似。下述主题提供了有关脚本部件文件的基本布局，同时也提供了关于创建注册信息和类型库的信息。

  • 脚本部件文件的内容   学习脚本部件所必需的 XML (可扩展标记语言) 元素。
  • 使用脚本部件向导   使用脚本部件向导来创建脚本部件文件。
  • 创建注册信息
    在脚本部件文件中创建在将其注册为 COM 部件时要使用的信息。
  • 说明方法   给脚本部件添加方法。
  • 说明属性   给脚本部件添加属性。
  • 说明事件   在脚本部件中定义和激活自定义事件。
  • 创建脚本部件类型库   为脚本部件创建类型库，以便能更方便地在宿主应用程序中使用该脚本部件。
  • 引用其他部件   包括对脚本中所需的对象的全局引用，或对提供脚本中所用常数的外部类型库的全局引用，或对指向没有硬编码到脚本中的字符串或值的资源的全局引用。
  • 引用同一个程序包中的其他脚本部件   当同一个程序包中包括多个脚本部件时，将一个脚本部件的功能嵌入到另一个脚本部件中。
  • 检查脚本部件文件中的错误   给脚本部件添加编译时的错误检查。
  • 脚本部件文件和 XML 一致性   学习 XML 的一致性。
