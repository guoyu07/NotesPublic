For 循环未初始化

在脚本中执行跳到For...Next 循环中间的结果。由于For...Next 循环计数必须被初始化所以产生错误。下面演示了一个For...Next 循环的正确结构。

For counter = start To end [Step step]
    [statements]
    [Exit For]
    [statements]
Next

要纠正该错误

  • 移去跳转至For...Next 循环的语句。
  • 确保For...Next循环包含了所有必须的部分。
