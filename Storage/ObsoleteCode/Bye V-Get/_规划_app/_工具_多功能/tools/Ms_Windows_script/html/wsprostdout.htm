StdOut 属性 (WScript)

显示当前脚本的只读输出流。

object.StdOut

参数

object
WScript 对象。

说明

StdOut 属性返回一个表示标准输出流的对象。只可在使用 CScript.exe 时访问 StdIn、StdOut 和 StdErr 流。如果试图在使用 WScript.exe 时访问这些流，则会出错。
