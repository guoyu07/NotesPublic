Exists 方法

表示某个键值是否存在于 WshNamed 对象中。

object.Exists(key)

参数

object
WshNamed 对象。
Key
表示 WshNamed 对象的参数的字符串值。

说明

Exists 方法返回布尔值。如果所需的参数是在命令行上指定的，则它返回 true（否则返回 false）。

示例

不妨在命令行中键入以下信息：

myScript.vbs /c:"WSH is a wonderful thing"

可以使用这两行 JScript 代码来查找以下命令行开关是否可用于启动脚本。

WScript.Echo(WScript.Arguments.Named.Exists("C"));
WScript.Echo(WScript.Arguments.Named.Exists("D"));
