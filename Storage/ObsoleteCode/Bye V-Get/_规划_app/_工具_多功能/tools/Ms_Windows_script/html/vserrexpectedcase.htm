需要 'Case'

创建Select...End Select 构造，但未包含任何Case语句。下面演示了Select...End Select 构造的正确使用方法。

Select Case testexpression
   [Case expressionlist-n
      [statements-n]] . . .
   [Case Else expressionlist-n
      [elsestatements-n]]
End Select

要纠正该错误

确保Select...End Select 构造包含所有必须的部分。
