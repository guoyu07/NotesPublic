Windows 脚本宿主  

AtEndOfLine 属性

返回一个指明是否已到达输入流行尾的布尔值。

object.AtEndOfLine

参数

object
StdIn 文本流对象。

说明

AtEndOfLine 属性中包含一个指明是否已到达输入流行尾的布尔值。如果流指针立即指向输入流行尾标记的前面，则 AtEndOfLine 属性返回 True，否则返回 False。StdIn、StdOut 及 StdErr 属性和方法仅在用 CScript.exe 运行脚本时可用。如果用 WScript.exe 运行该脚本就会出错。

示例

下面的示例将从键盘读取一行文本，并在该行结束时显示所敲入的内容。

[VBScript] 
Dim Input
Input = ""

Do While Not WScript.StdIn.AtEndOfLine
   Input = Input & WScript.StdIn.Read(1)
Loop
WScript.Echo Input
[JScript] 
var input = "";
while (!WScript.StdIn.AtEndOfLine)
{
   input += WScript.StdIn.Read(1);
}
WScript.Echo(input);
