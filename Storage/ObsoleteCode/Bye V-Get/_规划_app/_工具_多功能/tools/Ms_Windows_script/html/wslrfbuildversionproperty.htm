Windows 脚本宿主  

BuildVersion 属性

返回 Windows 脚本宿主的生成版本号。

Object.BuildVersion

参数

Object
WScript 对象。

说明

BuildVersion 属性返回一个整数值。

Windows 脚本宿主的完整版本号由产品的发行版本号，后跟生成版本号组成。例如，如果 Windows 脚本宿主的产品发行版本号为 5.6，而生成版本号为 6626，则完整版本号为 5.6.6626。

示例

下列代码返回 Windows 脚本宿主的完整版本号。它由产品的发行版本号和生成版本号组成。

WScript.Echo WScript.Version & "." & WScript.BuildVersion;
