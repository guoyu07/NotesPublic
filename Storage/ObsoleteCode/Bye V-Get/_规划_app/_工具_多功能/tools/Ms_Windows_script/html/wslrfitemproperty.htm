Item 属性 (WshNamed)

提供对 WshNamed 对象中项目的访问。

Object.Item(key)

参数

Object
WshNamed 对象
key
要检索的项目名称。

说明

Item 属性返回一个字符串。对于集合来说，它将根据指定的关键字返回一项。在命令行中输入参数时，如果要把字符串用引号括起来，那么可以在字符串参数中使用空格。参看以下示例：
myscript.vbs /stringarg:"This string has spaces"

WshNamed 集合中的引号将被删除。对于要出现在 WshNamed 集合中的参数，必须已在命令行中使用。如果该参数没有任何值（如简单变量或空字符串），则 Item 属性返回一个空字符串。从 Item 属性请求不存在的命名参数将导致错误。要查看变量是否存在，请使用 Exists 方法。

示例 1

在以下示例中，提供了两个用来运行脚本的命名参数。在脚本内部，代码可导致命名参数输出。Item 属性用于索引命名参数集。

' 在命令提示符处键入以下命令行运行该脚本。
myScript.vbs /c:arg1 /d:arg2

' 如果以下代码在脚本内部执行。
WScript.Echo WScript.Arguments.Named.Item("c")
WScript.Echo WScript.Arguments.Named.Item("d")

' 则生成以下输出。
arg1
arg2
