Windows 脚本部件  

脚本部件概述

Windows® 脚本部件是一个令人激动的新技术，使您可以通过易于使用的脚本语言（Microsoft® Visual Basic® Scripting Edition (VBScript) 和 Microsoft® JScript®）来创建功能强大的、可复用的 COM 部件。下述主题提供了脚本部件技术的概述，并说明了如何创建和使用脚本部件。

  • 脚本部件简介   了解何为脚本部件，以及使用脚本部件的优点。
  • 脚本部件是如何运作的   了解脚本部件所包含的技术，以及如何创建脚本部件。
