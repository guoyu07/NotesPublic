getVarDate 方法

返回 Date 对象中的 VT_DATE 值。

dateObj.getVarDate()

必选项 dateObj 参数为 Date 对象。

说明

当与 COM 对象、ActiveX® 对象或其他以 VT_DATE 格式接受和返回日期值的对象（例如 Visual Basic 和 VBScript）进行交互时，使用 getVarDate 方法。实际格式取决于地区设置，不随 JScript 变化。

要求

版本 3

请参阅

getDate 方法 | parse 方法

应用于： Date 对象
