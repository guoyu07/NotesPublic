需要 'With'

创建了一个With块，但是以关键字End而不是语句End With 结束。一个 With 块必须使用一个End With 语句结束。下面演示了With块的正确组成。

With object
   statements
End With

要纠正该错误

  • 如果使用关键字End 结束一个With块，将它改成End With语句。
  • 确保With控制结构了所有必须的部分。
