MonthName 函数

返回表明指定月份的字符串。

MonthName(month[, abbreviate])

参数

month
必选项。月份的数值定义。例如，一月是 1，二月是 2，以此类推。
Abbreviate
可选项。Boolean 值，表明月份名称是否简写。如果省略，默认值为 False，即不简写月份名称。

说明

下面的示例利用MonthName 函数为日期表达式返回月份的缩写：

Dim MyVar
MyVar = MonthName(10, True) ' MyVar 包含 "Oct"。
