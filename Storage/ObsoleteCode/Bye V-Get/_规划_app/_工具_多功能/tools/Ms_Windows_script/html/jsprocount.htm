Count 属性

返回一个集合或 Dictionary 对象中的项目数。只读属性。

object.Count

其中 object 总是“应用于”列表中一个项目的名称。

说明

下面这段代码说明了 Count 属性的用法：

[JScript]
function CountDemo()
{
   var a, d, i, s;                 // 创建一些变量。
   d = new ActiveXObject("Scripting.Dictionary");   
   d.Add ("a", "Athens");          // 添加一些关键字和项目。
   d.Add ("b", "Belgrade");
   d.Add ("c", "Cairo");
   a = (new VBArray(d.Keys()));    // 获取关键字。
   s = "";
   for (i = 0; i < d.Count; i++)   //遍历 dictionary。
   {
      s += a.getItem(i) + " - " + d(a.getItem(i)) + "<br>";
   }
   return(s);                      // 返回结果。
}
[VBScript]
Function ShowKeys
   Dim a, d, i, s   ' 创建一些变量。
   Set d = CreateObject("Scripting.Dictionary")
   d.Add "a", "Athens"   ' 添加一些关键字和项目。
   d.Add "b", "Belgrade"
   d.Add "c", "Cairo"
   a = d.Keys   ' Get the keys.
   For i = 0 To d.Count -1 ' 遍历数组。
      s = s & a(i) & "<BR>" ' 创建返回字符串。
   Next
   ShowKeys = s
End Function
