Windows 脚本部件简介

Windows® 脚本部件为您提供了用脚本创建功能强大、可复用的 COM 部件的简便方式。您可以使用任何支持 Microsoft® ActiveX® Scripting 接口的脚本语言来创建脚本部件。支持这些接口的脚本语言有： JScript、 Microsoft® Visual Basic® Scripting Edition (VBScript)、PERLScript、PScript 和 Python。

注意   关于 ActiveX Scripting 接口的详细信息，请参阅在 www.microsoft.com 的Microsoft Scripting Technologies Web 站点。

新的脚本部件技术支持公共类型的 COM 部件，如 Automation，并且可以使用 DHTML Behavior等外接程序加以扩展。

脚本部件:

  • 小巧高效。
  • 易于创建、维护和部署。
  • 提供了创建 COM 部件的能力。
  • 提供了访问大量系统服务的能力。

使用脚本部件可以为多种任务创建 COM 部件，如执行中间层业务逻辑、访问和操作数据库数据、给应用程序添加事务处理以及使用 DHTML Behavior给 Web 页添加交互效果等。
