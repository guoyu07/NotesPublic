变量未定义

在脚本开始处用Option Explicit 语句，后来又使用一个未被说明的变量名称。当使用Option Explicit 语句时，必须用Dim，Private，Public或者ReDim语句显式地说明所有的变量。

要纠正该错误

使用Option Explicit 语句时， 确保用Dim, Private, Public, or ReDim语句定义了所有的变量。
