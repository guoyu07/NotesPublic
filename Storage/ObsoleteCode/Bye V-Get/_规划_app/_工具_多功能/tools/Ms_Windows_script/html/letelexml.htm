Windows 脚本部件  

<?XML ?>

指明该文件应按 XML 进行语法分析。

<?XML version="version" standalone="DTDflag" encoding="encname" ?>

值

version
格式为 n.n 的字符串，指定该文件的 XML 级别。可以使用 1.0。
DTDflag
(可选项) 是一个 Boolean 值，指明该 XML 文件是否包括外部文档类型定义 (DTD) 的引用。脚本部件的 XML 文件不能包括这样的引用，因此这个属性的值始终为 “yes” 。
encname
(可选项) 描述 XML 文档所使用的字符集编码的字符串，该字符串包括 Microsoft® Internet Explorer 所支持的任何字符集。

说明

这个声明必须是文件的第一个元素（包括空格）。

如果有这个声明，则将使脚本部件编译程序采用严格的 XML 模式，这时元素类型和属性名称都是区分大小写的，属性值必须用单引号或双引号引起来，且分析所有的元素。如果没有包括这个声明，则编译程序将允许较为松散的语法。

通常，如果可能要用 XML 编辑器来编辑该脚本部件文件，则应该包括这个声明并遵循 XML 约定。

示例

<?XML version="1.0" standalone="yes" encoding="UTF-16" ?>
