Windows 脚本宿主  

对象

本节内容

WScript 对象
提供对 WSH 对象模型中的大多数对象、方法和属性的访问。
WshArguments 对象
赋予您访问整个命令行参数集合的权限，这些参数按照最初输入的顺序排列。
WshController 对象
显示用于创建远程脚本过程的 CreateScript() 方法。
WshEnvironment 对象
赋予您对 Microsoft Windows 系统环境变量集合的访问权限。
WshNamed 对象
提供对 WshArguments 对象内的命名命令行脚本参数的访问权限。
WshNetwork 对象
赋予您对计算机所连接的网络上的共享资源的访问权限。
WshRemote 对象
提供对远程脚本过程的访问权限。
WshRemoteError 对象
因脚本出错而使远程脚本（WshRemote 对象）终止时，显示可用的错误信息。
WshScriptExec 对象
提供有关用 Exec 运行的脚本的状态和错误信息，并提供对 stdIn、stdOut 和 stdErr 通道的访问。
WshShell 对象
赋予您对本地 Windows 外壳程序功能的访问权限。
WshShortcut 对象
允许您以编程方式创建快捷方式。
WshSpecialFolders 对象
允许您访问 Windows 特殊文件夹。
WshUnnamed 对象
提供对 WshArguments 对象内未命名的命令行脚本参数的访问权限。
WshUrlShortcut 对象
允许以编程方式创建 Internet 资源的快捷方式。
