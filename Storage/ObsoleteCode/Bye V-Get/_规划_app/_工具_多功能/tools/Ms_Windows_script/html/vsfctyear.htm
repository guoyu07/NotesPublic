Year 函数

返回一个代表某年的整数。

Year(date)

date 参数是任意可以代表日期的参数。如果 date 参数中包含 Null，则返回 Null。

说明

下面例子利用 Year 函数得到指定日期的年份：

Dim MyDate, MyYear
MyDate = #October 19, 1962#   ' 分派一日期。
MyYear = Year(MyDate)         ' MyYear 包含 1962。
