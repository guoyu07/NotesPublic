TypeName 函数

返回一个字符串，提供有关变量的 Variant 子类型信息。

TypeName(varname)

必选项的 varname 参数，可以是任何变量。

返回值

TypeName 函数返回值如下：

值 描述
Byte 字节值
Integer 整型值
Long 长整型值
Single 单精度浮点值
Double 双精度浮点值
Currency 货币值
Decimal 十进制值
Date 日期或时间值
String 字符串值
Boolean Boolean 值；True 或 False
Empty 未初始化
Null 无有效数据
<object type> 实际对象类型名
Object 一般对象
Unknown 未知对象类型
Nothing 还未引用对象实例的对象变量
Error 错误

说明

下面的示例利用 TypeName 函数返回变量信息：

Dim ArrayVar(4), MyType
NullVar = Null   ' 赋 Null 值。

MyType = TypeName("VBScript")   ' 返回 "String"。
MyType = TypeName(4)            ' 返回 "Integer"。
MyType = TypeName(37.50)        ' 返回 "Double"。
MyType = TypeName(NullVar)      ' 返回 "Null"。
MyType = TypeName(ArrayVar)     ' 返回 "Variant()"。

要求

版本 2
