实现 DHTML Behavior 脚本部件

Windows® 脚本部件提供了一种开销较小、易于维护的方式，可以创建用来实现 Microsoft® Internet Explorer 5.0 的 DHTML Behavior 的部件。通过使用脚本部件，脚本开发人员可以使用 Microsoft® Visual Basic® Scripting Edition (VBScript)、Microsoft® JScript® (ECMAScript) 或任何支持 Microsoft® ActiveX® Scripting 接口的第三方脚本语言来实现操作。

关于 DHTML Behavior 的一般信息，请访问 http://msdn.microsoft.com/workshop。

通过创建 Behavior 脚本部件，可以：

  • 通过将脚本部件的函数绑定到包含文档中产生的事件来说明操作，不管是为特定的元素，还是为 DHTML 文档 或 DHTML 窗口对象。
  • 定义布局，其中包括在调用该操作时插入到包含文档中的 HTML 文本。
  • 说明自定义属性方法，从而将元素的成员集合扩展到包含页面。
  • 说明自定义事件，脚本部件可以在任何时候激活这些事件。例如，脚本部件可以在更改了页面上的某个 DHTML 元素的内容后激活一个自定义事件。
  • 将脚本部件的函数绑定到某些标准事件，如载入文档或元素的内容被更改时。

可以在本节找到有关下述内容的信息：

  • 创建 Behavior 脚本部件   了解 Behavior 脚本部件文件中可以加入哪些元素，如何与包含文档共享信息，哪些 DHTML 特性可应用于 Behavior 脚本部件，以及在执行脚本时如何观察。
  • 在 Behavior 脚本部件中说明属性和方法   创建属性和方法，扩充 DHTML 元素中已经可用的属性和方法。
  • 在 Behavior 脚本部件中说明自定义事件   在脚本部件中为包含文档说明和激活事件。
  • Behavior 处理程序参考    Behavior 脚本部件中所使用的 XML 元素、属性和方法清单。
