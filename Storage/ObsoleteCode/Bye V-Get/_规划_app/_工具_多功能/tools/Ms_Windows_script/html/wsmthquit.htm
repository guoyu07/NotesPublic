Quit 方法

强制脚本随时停止执行。

object.Quit([intErrorCode])

参数

object
WScript 对象。
intErrorCode
可选。作为过程的退出代码而返回的整数值。如果未包括 intErrorCode 参数，则不返回任何值。

说明

Quit 方法可返回可选的错误代码。如果 Quit 方法是脚本中最终的指令（并且您无需返回非零值），则可忽略它，脚本将正常终止。

示例

下面的 JScript 代码片段退出执行并返回错误代码 1：

WScript.Quit (1);

// 本行代码从不执行。
var i = 0;
