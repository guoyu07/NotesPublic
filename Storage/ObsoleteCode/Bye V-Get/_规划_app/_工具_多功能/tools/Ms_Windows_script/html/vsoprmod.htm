Mod 运算符

两个数值相除并返回其余数。

result = number1 Mod number2

参数

result
任意数值变量。
number1
任意数值表达式。
number2
任意数值表达式。

说明

模数或余数。运算符执行 number1 除以 number2 操作（浮点数四舍五入为整数）并只返回余数作为 result。例如，在下面的表达式中， A（为 result）等于 5。

A = 19 Mod 6.7

如果任一表达式为 Null，则 result 也为 Null。任一表达式为 Empty 时按 0 来处理。

要求

版本 1
