Windows 脚本宿主基础

Microsoft® Windows® 脚本宿主 (WSH) 是一种与语言无关的脚本宿主，它可用于与 Windows 脚本兼容的脚本引擎。它为 Windows 32 位平台提供简单、功能强大而又灵活的脚本编写功能，允许您从 Windows 桌面和命令提示符运行脚本。

Windows 脚本宿主非常适合于非交互式脚本编写的需要，如脚本登录、脚本管理和计算机自动化。
