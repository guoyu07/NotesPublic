需要 'catch'

使用了处理异常的try 语句块，但没有些相应的 catch 语句。异常处理机制需要将那些可能失败的代码，连同当异常发生后不会执行的代码一起，包在一个 try 语句块中。发生异常时通过使用 throw 语句将异常从 try 语句块中抛出，并被 try 语句块外的一个或多个catch 语句抓获。

要改正此错误

  • 添加相应的 catch 语句块。
  • 试试使用 finally 语句块而不使用 catch语句块。
