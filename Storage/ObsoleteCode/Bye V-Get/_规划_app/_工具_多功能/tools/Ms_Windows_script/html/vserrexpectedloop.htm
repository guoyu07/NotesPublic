需要 'Loop'

创建了一个Do 循环结构，但未正常中断循环。Do 循环结构与Loop 语句相关联结束。下面演示了Do 循环结构的正确使用。

Do [{While | Until} condition]
   [statements]
   [Exit Do]
   [statements]
Loop

或者可以使用下面的语法：

Do
   [statements]
   [Exit Do]
   [statements]
Loop [{While | Until} condition]

要纠正该错误

确保Do循环结构包含了所有必须的部分。
