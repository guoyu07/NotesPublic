必须在一个类的内部定义

创建了一个 Property 过程（Property Let， Property Set，或 Property Get）， 但是在 Class 定义的外部进行了定义。只能在同一类里面创建 Property 过程。

要纠正该错误

确保所有的 Property Let、Property Set 和 Property Gets 在一个 Class 定义里面定义了。
