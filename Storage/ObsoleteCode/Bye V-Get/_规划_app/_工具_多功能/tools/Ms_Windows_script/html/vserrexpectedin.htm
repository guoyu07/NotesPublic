需要 'In'

创建一个For Each...Next 块，但在第一行未包括关键字In 。下面演示了For Each...Next 块的正确结构。

For Each element In group
   [statements]
   [Exit For]
   [statements]
Next [element]

要纠正该错误

确保For Each...Next 块包含了所有必须的部分。

For Each...Next 语句 | Do...Loop 语句 | Exit 语句 | For...Next 语句 | While...Wend 语句
