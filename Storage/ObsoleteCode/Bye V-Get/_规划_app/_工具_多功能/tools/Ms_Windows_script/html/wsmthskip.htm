Windows 脚本宿主  

Skip 方法

从输入文本流中读取字符时，跳过指定数量的字符。

object.Skip(characters)

参数

object
StdIn 文本流对象。
characters
表示要跳过的字符数的整数值。

说明

StdIn、StdOut 和 StdErr 属性与方法只在用 CScript.exe 主机可执行文件运行脚本时奏效。用 WScript.exe 运行脚本时将返回 "Invalid Handle"（句柄无效）错误。位置指针向前移动 characters 参数中指定的字符（字节）数。不能使用 Skip 方法在文件中向后跳过字符（不支持负字符数）。Skip 方法只限于在读取模式下打开文件（在向输出流写入时，不能跳过指定数量的字符）。

示例

下面的代码使用 Skip 方法跳过文本流中的第一个字符，从键盘读取字符，并将它写入 StdOut 文本流。

[VBScript] 
WScript.StdIn.Skip 1
Input = WScript.StdIn.ReadLine
WScript.StdOut.Write Input
[JScript] 
WScript.StdIn.Skip(1);
Input = WScript.StdIn.ReadLine();
WScript.StdOut.Write(Input);
