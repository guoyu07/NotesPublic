Windows 脚本宿主  

<description> 元素

标记在执行 ShowUsage 方法时或用 /? 命令行开关运行脚本时显示的说明性文本。

<description>
This section describes the script
</description>

说明

所作的说明可以比一行长。

如果不希望用法中出现引号，就不要在说明中包括它们。说明标记之间的所有文本都显示在用法列表中。其中包括制表符、新行和特殊字符。

<description> 元素类似于 <example> 元素，区别在于 <description> 元素出现在用法的开头，而 <example> 元素出现在结尾。

示例

下面的脚本将演示 <description> 元素的用法：

<runtime>
<description>
This script reboots a server
</description>
<!--...etc...-->
</runtime>
