使用脚本部件向导

Windows® 脚本部件程序包中包括了一个脚本部件向导，该向导可以自动完成创建脚本部件 .wsc 文件的过程。该向导将执行下述功能：

  • 创建 .wsc 文件并添加基本的 XML 元素。
  • 提示并创建注册信息。
  • 提示输入脚本部件要使用的接口处理程序类型。
  • 提示输入要说明的属性，并用指定的脚本语言创建必要的元素。
  • 提示输入要说明的方法，并用指定的脚本语言创建必要的元素。
  • 提示输入脚本部件可以使用的事件并创建事件元素的框架。
    注意   您可以从 Microsoft Scripting Technologies Web 站点下载和安装该脚本部件向导。

该向导将使用您提供的信息创建一个脚本部件的框架，并作为 .wsc 文件输出。请编辑该 .wsc 文件并添加您的脚本来创建脚本部件的功能。

注意   该向导所创建的脚本部件文件的开始包含 <?XML ?> 声明，表示该文件的内容必须是与 XML 一致的。详细信息请参阅脚本部件文件和 XML 一致性。
