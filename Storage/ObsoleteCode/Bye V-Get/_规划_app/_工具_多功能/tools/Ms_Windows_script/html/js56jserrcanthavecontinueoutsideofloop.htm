在循环外不能有 'continue'

试图在循环外使用 continue 语句。只有在下述语句内才可以使用 continue 语句：

  • do-while 循环，
  • while 循环，
  • for 循环，
  • for/in 循环。

要改正此错误

请确保 continue 语句位于下述语句的内部：

  • do-while 循环，
  • while 循环，
  • for 循环，
  • for/in 循环。
