需要 'Select'

创建了一个Select...End Select 结构，但未包含结束语句End Select 。下面演示了Select...End Select 结构的正确使用方法。

Select Case testexpression
   [Case expressionlist-n
      [statements-n]] . . .
   [Case Else expressionlist-n
      [elsestatements-n]]
End Select

要纠正该错误

确保Select...End Select结构包含了所有必须的部分。
