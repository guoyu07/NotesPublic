LoadPicture 函数

返回图片对象。

LoadPicture(picturename)

picturename 参数是字符串表达式，该表达式指明了要装入的图片文件的名称。

说明

可以由 LoadPicture 识别的图形格式有位图文件 (.bmp)、图标文件 (.ico)、行程编码文件 (.rle)、图元文件 (.wmf)、增强型图元文件 (.emf)、GIF (.gif) 文件和 JPEG (.jpg) 文件。

要求

版本 2
