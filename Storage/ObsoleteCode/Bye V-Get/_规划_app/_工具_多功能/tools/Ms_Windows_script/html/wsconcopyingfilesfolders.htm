复制文件和文件夹

文件系统的操纵（如复制文件和文件夹）需要用到文件系统对象 (FSO)。下面的脚本将演示如何使用 FSO 来复制文件和文件夹。

复制文件

下面的脚本将演示如何将文件从一个本地文件夹复制到另一个本地文件夹中。首先，脚本将创建一个文件系统对象。CopyFile 方法（文件系统对象的一个方法）执行文件复制操作。CopyFile 方法采用两个参数，即源文件和目标文件。

// JScript。
var FSO = WScript.CreateObject("Scripting.FileSystemObject");
FSO.CopyFile("c:\\COMPlusLog.txt"，"c:\\x\\");

' VBScript。
Dim FSO
Set FSO = CreateObject("Scripting.FileSystemObject")
FSO.CopyFile "c:\COMPlusLog.txt"，"c:\x\"

复制文件夹

下面的脚本将演示如何将一个本地文件夹中的内容复制到本地计算机上的另一个文件夹中。

注意   目标文件夹必须已经存在，该方法才能成功。有关如何用 WSH 创建目录的信息，请参阅 CreateFolder 方法。

首先，脚本将创建一个文件系统对象。CopyFolder 方法（文件系统对象的一个方法）执行文件夹复制操作。CopyFolder 方法采用两个参数，即源文件夹和目标文件夹。

// JScript。
var FSO = WScript.CreateObject("Scripting.FileSystemObject");
FSO.CopyFolder("c:\\x"，"c:\\y");

' VBScript。
Dim FSO
Set FSO = CreateObject("Scripting.FileSystemObject")
FSO.CopyFolder "c:\x"，"c:\y"
