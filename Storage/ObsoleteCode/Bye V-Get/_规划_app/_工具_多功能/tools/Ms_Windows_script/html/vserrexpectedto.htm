需要 'To'

创建一个For...Next 循环，但没有在第一行中包含关键字To。一个For...Next循环 的第一行必须包含关键字To 。下面演示了一个For...Next 循环的正确结构。

For counter = start To end [Step step]
    [statements]
    [Exit For]
    [statements]
Next

要纠正该错误

确保For...Next循环包含了所有必须的部分。
