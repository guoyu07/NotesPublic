Sqr 函数

返回数值的平方根。

Sqr(number)

number 参数可以是任意有效的大于或等于零的数值表达式。

说明

下面的示例利用 Sqr 函数计算数值的平方根：

Dim MySqr
MySqr = Sqr(4)   ' 返回 2。
MySqr = Sqr(23)   ' 返回4.79583152331272。
MySqr = Sqr(0)   ' 返回0。
MySqr = Sqr(-4)   ' 产生实时错误。
