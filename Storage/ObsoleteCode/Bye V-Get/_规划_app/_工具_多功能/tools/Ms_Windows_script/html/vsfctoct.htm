Oct

返回表示数字八进制值的字符串。

Oct(number)

number 参数是任意有效的表达式。

说明

如果 number 参数不是整数，则在进行运算前，将其四舍五入到最接近的整数。

如果 number 是 Oct 返回
Null Null。
Empty 零 (0)。
任意其他数字 最大可到 11 位的八进制字符。

用户也可以通过直接在数前加上 &O 表示八进制数。例如，&O10 为十进制数 8 的八进制表示法。

下面的示例利用 Oct 函数返回数值的八进制数：

Dim MyOct
MyOct = Oct(4)     ' 返回 4。
MyOct = Oct(8)     ' 返回 10。
MyOct = Oct(459)   ' 返回 713。
