不需要 'Next'

创建了一个For...Next 或者For Each...Next 构造，但没有包含For语句。例如，在试图去掉For...Next构造时保留了Next语句。下面演示了For...Next构造的正确用法。

For counter = start To end [Step step]
    [statements]
    [Exit For]
    [statements]
Next

要纠正该错误

  • 加上For语句 （循环控制语句）。
  • 去掉多余的Next语句。
