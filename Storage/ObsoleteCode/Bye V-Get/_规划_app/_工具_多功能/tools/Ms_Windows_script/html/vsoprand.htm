And 运算符

对两个表达式进行逻辑“与”运算。

result = expression1 And expression2

参数

result
任意数值变量。
expression1
任意表达式。
expression2
任意表达式。

说明

当且仅当两个表达式均为 True，则 result 为 True。如果任一表达式为 False，则 result 为 False。下表说明如何确定 result：
