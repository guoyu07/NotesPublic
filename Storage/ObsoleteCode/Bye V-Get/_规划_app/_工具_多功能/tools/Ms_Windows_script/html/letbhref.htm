Behavior 处理程序参考

使用 Behavior 处理程序，Microsoft® Internet Explorer 为 Behavior 脚本部件提供了通过自定义事件及其包含页面进行通信的手段，并且提供了访问包含文档的对象模型的手段。

下述元素、属性和方法是 Behavior 处理程序特有的。

  • <attach> 元素
  • <layout> 元素
  • element 属性
  • attachNotification 方法
  • createEventObject 方法
  • fireEvent 方法
