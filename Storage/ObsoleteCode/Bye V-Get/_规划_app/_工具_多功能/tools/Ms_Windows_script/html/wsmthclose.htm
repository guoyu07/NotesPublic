Close 方法

关闭文本流。

object.Close

参数

object
StdIn、StdOut 或 StdErr 文本流对象。

说明

StdIn、StdOut 和 StdErr 属性与方法只在用 CScript.exe 主机可执行文件运行脚本时奏效。如果用 WScript.exe 运行脚本，则会返回错误。不必关闭标准流；它们会在过程终止时自动关闭。如果关闭标准流，则要注意指向标准流的任何其他指针将变得无效。该方法旨在与 TextStream 对象兼容。
