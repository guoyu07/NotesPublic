Minute

返回 0 到 59 之间的一个整数（包括 0 和59），代表一小时内的某一分钟。

Minute(time)

time 参数是任意可以代表时间的表达式。如果 time 参数包含 Null，则返回 Null。

说明

下面的示例利用 Minute 函数返回小时的分钟数：

Dim MyVar
MyVar = Minute(Now)

要求

版本 1
