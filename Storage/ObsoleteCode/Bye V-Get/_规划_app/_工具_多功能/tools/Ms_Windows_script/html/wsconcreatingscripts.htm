Windows 脚本宿主  

创建可由 WSH 使用的脚本

Windows 脚本是一个文本文件。只要您用与 WSH 兼容的脚本扩展名（.js、.vbs 或 .wsf）保存脚本，就可用任何文本编辑器创建脚本。

最常用的文本编辑器（记事本）已经安装在您的计算机上了。还可以使用您所钟爱的 HTML 编辑器、Microsoft Visual C++ 或 Visual InterDev。

用记事本创建脚本

  1. 启动记事本。
  2. 编写脚本。作为示例，请键入 WScript.Echo("Hello World!");
  3. 用 .js 扩展名（而不是默认的 .txt 扩展名）保存该文本文件。例如，Hello.js。
  4. 导航到刚保存过的文件，并双击它。
  5. Windows 脚本宿主调用 JScript 引擎并运行您的脚本。在该示例中，会出现一个消息框，显示消息 "Hello World!"
