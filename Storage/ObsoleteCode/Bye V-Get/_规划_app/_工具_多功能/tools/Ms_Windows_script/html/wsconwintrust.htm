CryptoAPI 工具

CryptoAPI 工具用于创建和验证 *.exe、*.cab、*.dll、*.ocx 和脚本（*js、*.vbs 和 *.wsf）文件中的数字签名。

CryptoAPI 工具用来对用于 Microsoft® Authenticode® 的文件进行数字签名，并用来查看和管理证书、证书吊销列表 (CRL) 和证书信任列表 (CTL)。有关 CrptoAPI 工具的详细信息，请参阅 (http://msdn.microsoft.com/library/psdk/crypto/portalapi_3351.htm?RLD=290) 的CryptoAPI 起始页。
