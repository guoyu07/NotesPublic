Windows 脚本宿主  

WriteBlankLines 方法

向输出流发送指定数量的空行（换行符）。

object.WriteBlankLines(intLines)

参数

object
StdOut 或 StdErr 文本流对象。
intLines
表示要写入流的空行数的整数值。

说明

在不提供 strText 参数的情况下调用 WriteLine 方法等同于调用 WriteBlankLines(1)。StdIn、StdOut 和 StdErr 属性与方法只在用 CScript.exe 主机可执行文件运行脚本时奏效。用 WScript.exe 执行脚本时将返回 "Invalid Handle"（句柄无效）错误。

示例

下面的代码演示 WriteBlankLines 方法。

[VBScript] 
Dim StdIn, StdOut
Set StdIn = WScript.StdIn
Set StdOut = WScript.StdOut

Do While Not StdIn.AtEndOfStream
     str = StdIn.ReadLine
     StdOut.Write "Line " & (StdIn.Line - 1) & ": " & str
     StdOut.WriteBlankLines 1
Loop
[JScript] 
var stdin = WScript.StdIn;
var stdout = WScript.StdOut;

while (!stdin.AtEndOfStream)
{
     var str = stdin.ReadLine();
     stdout.Write("Line " + (stdin.Line - 1) + ": " + str);
     stdout.WriteBlankLines(1);
}
