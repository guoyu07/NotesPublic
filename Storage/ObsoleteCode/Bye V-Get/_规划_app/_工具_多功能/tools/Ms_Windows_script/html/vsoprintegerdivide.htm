整除运算符 (\)

两个数相除并返回以整数形式表示的结果。

result = number1\number2

参数

result
任意数值变量。
number1
任意数值表达式。
number2
任意数值表达式。

说明

在除法操作前，数值表达式四舍五入为 Byte、Integer 或 Long 子类型表达式。

如果任何表达式为 Null，result 也是 Null。任何表达式为 Empty 时，则按 0 处理。

要求

版本 1
