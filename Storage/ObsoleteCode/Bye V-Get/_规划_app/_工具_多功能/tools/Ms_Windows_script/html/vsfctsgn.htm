Sgn 函数

返回表示数字符号的整数。

Sgn(number)

number 参数可以是任意有效的数值表达式。

返回值

Sgn 函数有如下返回值：

如果 number 为 Sgn 返回
大于零 1
等于零 0
小于零 -1

说明

number 参数的符号决定 Sgn 函数的返回值。

下面的示例利用 Sgn 函数决定数值的符号：

Dim MyVar1, MyVar2, MyVar3, MySign
MyVar1 = 12: MyVar2 = -2.4: MyVar3 = 0
MySign = Sgn(MyVar1)   ' 返回 1。
MySign = Sgn(MyVar2)   ' 返回 -1。
MySign = Sgn(MyVar3)   ' 返回 0。
