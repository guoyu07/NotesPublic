source 属性

返回正则表达式模式的文本的复本。只读。

rgExp.source

rgExp 参数是正则对象。它可以是变量名或文字。

下面的例子举例说明了 source 属性的用法：

function SourceDemo(re, s){
   var s1;
   // 测试字符串中是否存在正则表达式。
   if (re.test(s))
      s1 = " contains ";
   else
      s1 = " does not contain ";
   // 获得正则表达式自己的文本。
   return(s + s1 + re.source);
}
