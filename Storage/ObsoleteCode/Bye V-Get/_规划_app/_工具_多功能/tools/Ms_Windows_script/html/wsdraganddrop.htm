WSH 拖放支持

可以将文件拖到 WSH 脚本上。文件名在命令行上转化为参数。这些文件名可以显示在一个列表中，以便通过任意脚本对象来操纵文件。

显示脚本的参数列表

  1. 创建文件，并使用带有脚本扩展名的名称（如 DragDrop.vbs）。
  2. 将代码添加到该脚本文件中，例如： 
    Set objArgs = WScript.Arguments
For I = 0 to objArgs.Count - 1
WScript.Echo objArgs(I)
Next
  3. 将该文件保存到硬盘上。
  4. 将任意文件拖放到已保存的文件上。在本示例中，文件名将回显在屏幕上。

可以拖放到一个脚本的文件数受系统最大命令行长度的限制。如果被拖放的所有文件名的字符总数超过此长度限制，则拖放操作无法执行。
