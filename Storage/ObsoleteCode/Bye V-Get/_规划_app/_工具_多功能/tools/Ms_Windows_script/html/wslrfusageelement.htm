Windows 脚本宿主  

<usage> 元素

允许用户替代默认的用法显示。

<usage>
This section describes the script
</usage>

说明

<usage> 元素与 <example> 和 <description> 元素类似，它包含开发人员可以放入用法中的文本。区别在于，如果 <usage> 元素存在，<runtime> 元素中的所有其他内容都会被忽略，只显示 <usage> 元素中的文本。因此，您可以完全替代用法显示。

注意   <usage> 元素应始终用 <runtime> 元素括起来。

示例

下面的脚本将演示 <usage> 元素的用法：

<job>
<runtime>
<named name="arg1" helpstring="the first arg"/>
<usage>
Your usage text goes here.
</usage>
</runtime>
<script language="vbscript">
WScript.Arguments.ShowUsage
</script>
</job>

这将生成以下结果：

此处是您的用法文本。
