脚本和 Windows 自动化

计算机是一种很了不起的工具，它能执行很多重复性任务。但是，如果需要执行一系列重复性任务，该怎么办呢？答案便在脚本编写中。

什么是脚本？

脚本是用脚本语言（如 JScript 和 VBScript）编写的程序。可选脚本语言包括 Rexx、Python 和 Perl。与编程语言（如 C++ 和 Visual Basic）相比，脚本语言更适合于创建短小的应用程序以便快速解决小问题。

Windows 自动化

在许多情况下，脚本用于实现手动任务的自动化，这一点很象宏。脚本非常适合于：

  • 操纵 Windows 环境
  • 运行其他程序
  • 使登录过程自动化
  • 向应用程序发送按键顺序

例如，如果您有几个类似的任务，就可以编写可处理所有任务的通用脚本。

可以编写用于启动操作以响应事件的脚本。可以编写保持对事件连续计数并只在满足一定条件时才触发某些操作的脚本。

脚本还可用于非重复性的任务。如果某个任务需要依次执行许多操作，就可以通过编写脚本将这一系列任务简化为一个任务。
