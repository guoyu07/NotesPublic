Timer 函数

返回午夜 12 时以后已经过去的秒数。

Timer

说明

下面的例子使用 Timer 函数来确定 For...Next 循环 N 次所需的时间： 

Function TimeIt(N)
   Dim StartTime, EndTime
   StartTime = Timer
   For I = 1 To N
   Next
   EndTime = Timer
   TimeIt = EndTime - StartTime
End Function
