Windows 脚本宿主  

FullName 属性（WScript 对象）

返回主机可执行文件（CScript.exe 或 WScript.exe）的完全限定路径。

object.FullName

参数

object
WScript 对象。

说明

FullName 属性是代表主机可执行文件的完全限定路径的只读字符串。

示例

下面的 JScript 代码使用 FullName 属性。

WScript.Echo (WScript.FullName);

生成如下输出：

C:\WINNT\System32\cscript.exe
