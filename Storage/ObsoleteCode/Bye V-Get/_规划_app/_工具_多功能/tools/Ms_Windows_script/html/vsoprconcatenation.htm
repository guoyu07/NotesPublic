连接运算符 (&)

强制两个表达式进行字符串连接。

result = expression1 & expression2

参数

result
任意变量。
expression1
任意表达式。
expression2
任意表达式。

说明

任一 expression 不是字符串时，它将被转换为 String 子类型。如果两个表达式都为 Null，result 也为 Null。然而，如果仅有一个 expression 为 Null，则它和其他表达式连接时，按零长度字符串("") 处理。任何表达式为 Empty 时也按零长度字符串处理。

要求

版本 1
