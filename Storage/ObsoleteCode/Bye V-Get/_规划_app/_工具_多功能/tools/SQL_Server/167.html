（2） RAND( ) 的一般用法

生成在一定范围的随机数

--   0~9999
(CONVERT(int,RAND()*100000)%10000)

--   -9999~9999
CASE 
WHEN (CONVERT(int,RAND()*100)%2=1 THEN
  (CONVERT(int,RAND()*100000)%10000*-1)
ELSE (CONVERT(int,RAND()*100000)%10000)
END

--   A~Z
CHAR((CONVERT(int,RAND()*1000)%26)+65)  --65 is A
