taintEnabled版本：DHTML Object Model　返回值：有
语法：
bEnabled = navigator . taintEnabled ()
参数：
无
返回值：
bEnabled :布尔值(Boolean)。false | true
false :不支持( data tainting )。
true :支持( data tainting )。
说明：
返回( data tainting )是否被激活。
因为 IE5 及其更早版本不支持( data tainting )，所以总是返回 false 。
示例：
var bEnabled = navigator.taintEnabled();
