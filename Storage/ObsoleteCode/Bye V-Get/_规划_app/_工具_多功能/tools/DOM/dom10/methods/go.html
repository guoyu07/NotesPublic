go版本：DHTML Object Model　返回值：无
语法：
history . go ( vLocation )
参数：
vLocation : 必选项。
整数(Integet) : 指定要到达的历史( history )列表里的 URL 地址的相对距离。
字符串(String) : 指定要到达的历史( history )列表里的 URL 的完整或部分地址。
返回值：
无
说明：
载入历史( history )列表里的一个 URL 地址。
假如用户试图跨越历史( history )列表的最后一项或最前一项，则会停留在当前的 URL 地址而不会发生错误。
示例：
window.history.go(-1);
