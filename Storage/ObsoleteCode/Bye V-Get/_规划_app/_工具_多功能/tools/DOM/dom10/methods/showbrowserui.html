ShowBrowserUI版本：DHTML Object Model　返回值：有
语法：
vReturn = external . ShowBrowserUI ( sUI , null )
参数：
sUI : 必选项。字符串(String)。指定一种浏览器对话框。LanguageDialog | OrganizeFavorites
LanguageDialog :打开「选择语言」对话框。
OrganizeFavorites :打开「整理收藏夹」对话框。
null : 必选项。 null 值。
返回值：
vReturn : 返回值由对话框决定。可能是任何类型的变量。
说明：
打开指定的一种浏览器对话框。
