urns版本：DHTML Object Model　返回值：有
语法：
arrObjects = object . urns ( sUrn )
参数：
sUrn : 必选项。字符串(String)。指定行为( behavior ) 的通用资源名称( URN )。
返回值：
arrObjects : 数组(Array)。成功则返回对象集合，否则返回 null 。
说明：
返回绑定了指定行为( behavior) 的一组对象的引用。
使用 length 属性可以获取 arrObjects 包含的对象数目。使用 item 方法可以获取 arrObjects 中某个对象的具体细节。
示例：
var coll = document.all.urns("rdl:demo");
