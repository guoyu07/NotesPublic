show版本：DHTML Object Model　返回值：无
语法：
popup . show ( iX , iY , iWidth , iHeight , oElement )
参数：
iX : 必选项。整数(Integet)。指定弹出窗口( popup )左上角横坐标。单位为像素( px )。
iY : 必选项。整数(Integet)。指定弹出窗口( popup )左上角纵坐标。单位为像素( px )。
iWidth : 必选项。整数(Integet)。指定弹出窗口( popup )宽度。单位为像素( px )。
iHeight : 必选项。整数(Integet)。指定弹出窗口( popup )高度。单位为像素( px )。
oElement : 可选项。对象(Element)。指定 iX 和 iY 坐标所相对于的对象。如果忽略此参数，则默认相对于桌面(Desktop)。
返回值：
无
说明：
显示弹出窗口( popup )。
