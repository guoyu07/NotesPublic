AddFavorite版本：DHTML Object Model　返回值：无
语法：
window . external . AddFavorite ( sURL , sTitle )
参数：
sURL : 必选项。字符串(String)。指定要加入收藏夹的 URL 地址。
sTitle : 可选项。字符串(String)。指定加入收藏夹后显示用的标题名称。
返回值：
无
说明：
提供一个对话框让用户将指定的 URL 加入收藏夹。
使用此方法等同于用户从浏览器的「收藏」菜单内选择「 添加到收藏夹...」。
此方法不支持HTML应用程式( HTA )。
示例：
window.external.AddChannel(window.location.href,window.document.title);
