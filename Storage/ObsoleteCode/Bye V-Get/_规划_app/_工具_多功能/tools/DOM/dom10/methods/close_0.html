close版本：DOM1　返回值：无
语法：
window . close ()
参数：
无
返回值：
无
说明：
关闭当前浏览器窗口或HTML应用程式( HTA )。
你关闭窗口的步骤决定了用户是否会得到确认提示。假如你对一个不是用脚本打开的窗口调用此方法，用户会看到一个确认对话框。这同样会发生在你用此方法离开当前最后一个 Microsoft® Internet Explorer 运行的实例时。当你用此方法关闭HTML应用程式( HTA )时则不会被提示。因为此种情况下，程序是被信任的并遵循另一个不同的安全模型。
当文档内的对象上的事件( event )激发的函数调用 close 方法时， window 对象的 close 方法被使用。而当文档内的对象上的事件( event )直接使用 close 方法时， document 对象的 close 方法被使用。
