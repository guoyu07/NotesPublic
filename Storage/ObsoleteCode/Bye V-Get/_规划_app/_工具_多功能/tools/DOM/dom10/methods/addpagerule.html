addPageRule版本：DHTML Object Model　返回值：有
语法：
plNewIndex = styleSheet . addPageRule ( sSelector , sStyle , iIndex )
参数：
sSelector : 必选项。字符串(String)。为新的 page 对象指定选择符(Selector)。
sStyle : 必选项。字符串(String)。为新的 page 对象指定样式信息。使用内联(Inline)样式定义的格式。如： "color : blue" 。
iIndex : 必选项。整数值(Integer)。指定新的 page 对象在 pages 集合里的位置。默认值为 -1 。即加入到 pages 集合的最后。
返回值：
plNewIndex : 此返回值为保留专用值。请勿使用。
说明：
为一个样式表单建立一个新的 page 对象。
每个 page 对象的样式定义对应于文档中的一个 @page 规则。
