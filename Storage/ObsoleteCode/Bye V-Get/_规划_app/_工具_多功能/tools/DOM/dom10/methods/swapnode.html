swapNode版本：DHTML Object Model　返回值：有
语法：
oElement = object . swapNode ( oNode )
参数：
oNode :必选项。对象(Element)。用于与 object 交换的对象。
返回值：
oElement :对象(Element)。返回调用本方法的对象的引用。
说明：
交换在文档结构中 object 与 oNode 的位置。
当交换发生时，所有与之相关的属性内容都将被交换。
oNode 与 object 必须已经存在。
在运行时使用此方法是可以的。在对象的关闭标签被解析前，文档空间可能不能被呈递。
示例：
document.body.children(2).swapNode(document.body.children(1));
