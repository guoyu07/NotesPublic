AddChannel版本：DHTML Object Model　返回值：无
语法：
window . external . AddChanne ( sURL )
参数：
sURL : 必选项。字符串(String)。指定要安装的 CDF 文件的 URL 地址。 CDF 文件是频道定义格式( Channel Definition Format )文件。
返回值：
无
说明：
显示一个对话框允许用户决定是否安装指定的频道( CDF )。
假如出错，会有错误对话框出现。你可以通过使用 onerror 事件禁止该对话框可能的出现。
此方法是用于 Microsoft® Active Channel® 内容发布预定的。Microsoft® Internet Explorer 4. 0及更高版本支持 Active Channel 技术。
此方法不支持HTML应用程式( HTA )。
示例：
window.external.AddChannel("http://www.dhtmlet.com/rdl_dom.cdf");
