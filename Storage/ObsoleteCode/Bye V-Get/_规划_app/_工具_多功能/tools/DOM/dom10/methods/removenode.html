removeNode版本：DHTML Object Model　返回值：有
语法：
oElement = object . removeNode ( bRemoveChildren )
参数：
bRemoveChildren : 可选项。布尔值(Boolean)。false | true
false : 默认值。删除 object 时保留 object 的 childNodes 集合。即其所有子对象。
true : 删除 object 时也删除 object 的 childNodes 集合。即其所有子对象。
返回值：
oElement : 对象(Element)。返回被删除对象的引用。
说明：
从文档树中删除 object 。
在运行时使用此方法是可以的。在对象的关闭标签被解析前，文档空间的其他区域不能被呈递。
