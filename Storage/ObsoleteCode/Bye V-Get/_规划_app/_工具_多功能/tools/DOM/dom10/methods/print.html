print版本：DHTML Object Model　返回值：无
语法：
window . print ()
参数：
无
返回值：
无
说明：
打印当前窗口内的文档。
调用此方法等同于从 Microsoft® Internet Explorer 文件菜单内选择「打印」。此方法将激活系统打印对话框供用户选择打印设置。
当用户点击「确定」时，将依次发生下列事件：
  1. onbeforeprint 事件被触发。
  2. 文档被打印。
  3. onafterprint 事件被触发。
当并非所有页面信息在任何时候总是可见的情况下， onbeforeprint 和 onafterprint 事件是非常有用的。使用 onbeforeprint 事件可以检测文档以保障打印时所有页面信息都是可见的。使用 onafterprint 事件可以恢复文档到它的最初状态。
