createStyleSheet版本：DHTML Object Model　返回值：有
语法：
oStylesheet = document . createStyleSheet ( sURL , iIndex )
参数：
sURL : 可选项。字符串(String)。指定怎样将样式规则加入文档。假如是一个文件名，样式信息将作为 link 对象加入。假如包含样式信息，则作为 style 对象加入。
iIndex : 可选项。整数值(Integer)。指定加入的样式信息在 styleSheets 集合里的索引序号。默认值为加入到 styleSheets 集合的最后。
返回值：
oStylesheet : 对象(Element)。返回一个 styleSheet 对象。
说明：
为文档建立样式规则。
