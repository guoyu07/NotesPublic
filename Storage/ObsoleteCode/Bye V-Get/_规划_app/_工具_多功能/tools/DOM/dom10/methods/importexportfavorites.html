ImportExportFavorites版本：DHTML Object Model　返回值：无
语法：
window . external . ImportExportFavorites ( bImportExport , sImportExportPath )
参数：
bImportExport : 必选项。布尔值(Boolean)。true | false
true : 导入请求。
false : 导出请求。
sImportExportPath : 可选项。字符串(String)。指定导入或导出的 URL 地址。假如忽略此参数，一个打开文件对话框会被显示。
返回值：
无
说明：
导入或导出 Microsoft® Internet Explorer 的收藏夹。
在导入或导出开始前需要确认。
导出时收藏夹所有内容被上传到服务器。建议将服务器设置为接受更新前删除已存在收藏内容。
从服务器导入的收藏和客户端已存在的收藏内容合并。服务器上的删除不会波及到客户端。
此方法不支持HTML应用程式( HTA )。
