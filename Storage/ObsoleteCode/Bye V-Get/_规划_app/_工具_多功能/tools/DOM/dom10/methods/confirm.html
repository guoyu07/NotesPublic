confirm版本：DHTML Object Model　返回值：有
语法：
bConfirmed = window . confirm ( sMessage )
参数：
sMessage : 可选项。字符串(String)。指定在对话框内要被显示的信息。如果忽略此参数，将不显示任何信息。
返回值：
bConfirmed : 布尔值(Boolean)。false | true
false :用户点击了对话框的「取消」。
true :用户点击了对话框的「确定」。
说明：
显示一个包含可选信息以及「确定」和「取消」按钮的确认对话框。
此对话框为模式对话框，其标题栏文字不可以被改变。
示例：
var bConfirm=window.confirm("Dream is my reality .");
