add版本：DHTML Object Model　返回值：有
语法：
oNamespace = namespaces . add ( sNamespace , sUrn , sUrl )
参数：
sNamespace : 必选项。字符串(String)。指定 namespace 的名字。
sUrn : 必选项。字符串(String)。指定 namespace 的通用资源名称( URN )。
sUrl : 可选项。字符串(String)。指定要导入 namespace 的元素行为( Element Behavior )的 URL 地址。指定此参数等同于使用 doImport 方法。
返回值：
oNamespace : 对象(Element)。返回 namespace 对象。
说明：
建立一个新的 namespace 对象并加入 namespaces 集合。
如果同名的 namespace 已经存在，它将被覆盖。
示例：
document.namespaces.add("rdl","rdl:demo","rdl_namespace_demo.htc");
