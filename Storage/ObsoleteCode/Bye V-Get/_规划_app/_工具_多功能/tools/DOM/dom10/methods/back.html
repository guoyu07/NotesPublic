back版本：DHTML Object Model　返回值：无
语法：
history . back ( iDistance )
参数：
iDistance : 可选项。整数(Integet)。指定要后退的 URL 地址数目。假如不提供此参数，将载入历史( history )列表里的上一个 URL 地址。
返回值：
无
说明：
载入历史( history )列表里的一个 URL 地址。
此方法作用等同于用户点击浏览器的「后退」按钮。也等同于 history.go(-1) 。假如用户已经到达历史( history )列表的最后一项，则会停留在当前的 URL 地址而不会发生错误。
示例：
window.history.back(1);
