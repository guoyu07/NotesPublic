# Using Docker
## ShadowSocks
```
sh$ docker pull oddrationale/docker-shadowsocks
sh$ docker run -d -p ${PORT OUTSIDE}:9527 oddrationale/docker-shadowsocks -s 0.0.0.0 -p 9527 -k ${PASS WORD} -m aes-256-cfb
```
